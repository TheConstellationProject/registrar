Constellation Network

The Constellation Network (ConstellationNet) is a place to securely host IP services away from the wider internet. Expensive domain prices, security threats, and address shortages create a less than ideal situation to host your application. ConstellationNet fixes this problem. With free domains, custom TLDs, and no shortage of IPv4 addresses with native IPv6 coming soon, there's no cost for addressing.

Technology

The network operates as a internet-linked intranet. We use ZeroTier SDN to provide intranet connectivity. Please note that this service DOES NOT provide internet connectivity. Detailed installation instructions and current downloads can be found on their website. https://www.zerotier.com/download/ Our ZeroTier network is private, and all devices must be registered prior to use via the form below.

Domains

We operate a non-recursive authoritative DNS server. While you are not required by the network to use ours as your primary DNS server, domains will not resolve any other way. The servers are located at 172.16.X.X All domains are provided free of charge. To register yours, use the domain portal.
