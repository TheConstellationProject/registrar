Arachnid
Network

A network for the privacy-minded, allowing users to host any TCP/IP based service on a software defined network.

About the Network

Arachnid aims to provide what the internet doesnt. Custom TLDs, no shortage of IP addresses, and baked in redundancy. Best of all, everything is free.

Flexible

Bring existing applications, or develop something new! Arachnid works with any TCP/IP service or protocol.

Professional Service

Utilizing ZeroTier's global network of planet servers and UDP hole punching whenever possible, speed is never a concern.

Extensible

With extensive platform support, you will be hard pressed to find a system that isn't compatible with out network.

Software Defined

Software Defined Networking (SDN) is the act of separating networking protocols from their infrastructure and implementing them using software instead. The network can run as a process in the background, exists solely in software, and thus can run on virtually any hardware platform.

Ultimate Freedom

We care about developing a safe, secure space to host your service. What you choose to do with the network is not our concern. Unlimited bandwidth, global connectivity, and transport encryption are all part of Arachnid's core. Additionally, Arachnid is 100% free. All domains and addresses come at no cost to the user.

Plans

All plans are free. In order to keep routing tables small, we do ask users to let us know the reason for large address blocks.
