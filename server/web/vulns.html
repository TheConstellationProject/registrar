Arachnid Network

Vulnerability Program

Arachnid is now open to a private vulnerability assessment by those in the "spiderdev" keybase group ONLY.

Scope:

  • Arachnid client and controller webapp. All routes past arachnid.cc/network. (arachnid.cc/network/search, etc)
  • arachnid.cc host. (45.63.84.10) All ports and services. Just don't take it down.

Exemptions: The following items are NOT in scope. Unauthorized access or pentesting is prohibited by US law.

  • ANY Subdomain of arachnid.cc EXCEPT for www
  • Our mail server
  • Any service or server that is not notated as in scope.
  • DoS attacks of any kind.
  • ANY Internal service. (172.16.0.0/16)

All reports should be sent via keybase to audit0r. Please report the vulnerability or bug within 2 hours of first discovery. We aren't expecting a full report, just let us know what the vuln is and steps to reproduce.
